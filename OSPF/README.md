# OSPF Labs

This directory contains my hands-on Cisco Packet Tracer labs focused on Open Shortest Path First (OSPF), completed as part of my CCNA studies.

## Objectives

These labs are designed to develop practical experience with:

- Configuring single-area OSPF
- Configuring multi-area OSPF
- Establishing OSPF neighbor adjacencies
- Verifying OSPF operation
- Understanding OSPF route selection
- Manipulating OSPF interface costs
- Configuring passive interfaces
- Advertising default routes
- Route summarization
- OSPF authentication
- Troubleshooting common OSPF issues

## Verification Commands

Common commands used throughout these labs include:

```text
show ip ospf
show ip ospf interface
show ip ospf neighbor
show ip ospf database
show ip route ospf
show ip protocols
show running-config | section router ospf
```
