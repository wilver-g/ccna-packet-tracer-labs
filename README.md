# CCNA Packet Tracer Labs

A collection of my Cisco CCNA Packet Tracer labs completed while studying for the Cisco Certified Network Associate (CCNA) certification.

This repository serves as a portfolio of my hands-on networking practice, covering fundamental networking concepts, routing and switching technologies, network security, and troubleshooting.

---

## 📚 Topics Covered

- VLANs
- Inter-VLAN Routing
- Trunking (802.1Q)
- EtherChannel (LACP & PAgP)
- Spanning Tree Protocol (STP)
- Rapid Spanning Tree Protocol (RSTP)
- OSPF
- Static Routing
- Default Routes
- ACLs (Standard & Extended)
- DHCP
- NAT / PAT
- HSRP
- Port Security
- DHCP Snooping
- Dynamic ARP Inspection (DAI)
- IPv6
- Network Device Management
- Troubleshooting Labs
